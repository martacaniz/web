Sitio web de Ejemplo para Mentores

Sitio para que los mentores lo utilicen como ejemplo cuando desarrollen sus propios sitios.

Edite el código según sea necesario para cumplir con los requisitos de su página.

Plan de desarrollo recomendado usando los Recursos de Programación

:
  1. Cree una cuenta de GitHub como se explica en el Tutorial de GitHub.
    • Haga clic en el botón disponible en la pagina "Recursos de Programación"
  2. Utilice el código (botón 1b) uno para hacer que un sitio html simple esté disponible en línea.
  3. Reemplace el código de su sitio con el del sitio de ejemplo (botón 2b).
  4. Personaliza su sitio editando el código.
  5. Decida qué características del sitio "MayaLink version html" (botón 3a) le gustaría incorporar en su sitio.
  6. Determine qué código se necesita para lograr esto revisando el código (botón3b).
  7. Edite su sitio según sea necesario para agregar las funciones que desee.
